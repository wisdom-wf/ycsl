# 变更日志 (CHANGELOG)

宜川县水利工程运行管理平台 — 前端项目

## [v1.5.0] — 2026-04-05

### 新增功能

**视频监控模块**

- 左侧面板新增视频监控区域，支持旋转加载动效（含科技感扫描线）
- 刘庄水库、崖底水库：加载 3 秒后显示"视频信号丢失"故障状态，含故障扫描线动画和重试按钮
- 钟楼寺水库、木头沟水库：加载 3 秒后展示实景监控截图，叠加 LIVE 标记、实时时间戳和四角边框装饰
- 图片资源路径：`/ycsl/video/zhonglousi.jpg`、`/ycsl/video/mutougou.jpg`

## [v1.4.0] — 2026-04-04

### 新增功能

**天气预警系统**

- 当 24h 降水量超过 20mm 或出现雷阵雨、暴雨、沙尘暴等灾害性天气时，整个标题区自动触发预警效果
- 支持四级预警等级：蓝色（一般）、黄色（注意）、橙色（警告）、红色（严重）
- 预警触发时标题区背景色、边框、文字颜色随等级变化，并显示预警标签和具体预警内容
- 标题区添加 CSS 呼吸灯动画，视觉上持续闪烁提示

## [v1.3.0] — 2026-04-04

### 新增功能

**实时天气接口对接**

- 对接中国天气网 API，宜川县城市代码 `101110304`
- 通过 Nginx 反向代理解决浏览器跨域问题，代理路径：`/weather-api/`
- 左上角实时展示：温度、天气状况、风向风速、湿度、24h 降水量、AQI 空气质量指数
- 天气数据每 30 分钟自动刷新，失败时显示"--"占位

## [v1.0.0] — 2026-03-24

### 初始版本发布

**监测信息驾驶舱**

- 三列布局：左侧信息面板（20%）+ 中部地图（60%）+ 右侧图表（20%）
- 左侧面板：水库选择下拉框、水库特征数据、实时监测指标、责任人信息（行政/技术/巡查）
- 中部地图：宜川县地图图片，叠加沿同心圆轨迹流动的亮线特效
- 右侧图表：近期降雨过程（柱状图）、库水情监测（折线图）、渗透压力监测（梯形剖面图）
- 顶部标题区：实时时钟（每秒更新）、天气信息展示

**运维信息驾驶舱**

- 三列布局与监测信息驾驶舱一致
- 左侧：区县运维情况（圆形进度条）、运维综合统计、问题处理统计、责任人信息
- 中部：顶部三个指标卡片 + 宜川县地图
- 右侧：区县水库运行信息、问题处理情况（饼状图）、库区水情监测、"查看工单"按钮

**工单管理弹窗**

- 点击"查看工单"按钮弹出工单列表
- 支持工单状态筛选（全部/待处理/处理中/已完成）
- 工单详情展示：编号、类型、水库、描述、负责人、创建时间、状态

**水库数据（本地模拟）**

- 四个水库：刘庄水库、钟楼寺水库、崖底水库、木头沟水库
- 每个水库包含：特征数据、实时监测指标、责任人信息（含真实姓名和联系电话）
- 切换水库时所有面板数据同步更新

**部署**

- 生产环境：`https://www.wisdomdance.cn/ycsl/`
- 服务器：43.153.213.134（Nginx 1.24.0）
- Nginx 配置：`/ycsl/` 路径 location 块，支持 SPA 前端路由
