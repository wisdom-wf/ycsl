### Bug 修复

**地图标注点漂移问题（彻底修复）**

- **根本原因**：地图图片（HTML `<img>`）与标注文字（CSS `position: absolute` 的 `<div>`）属于不同布局层，页面缩放时各自独立缩放，导致相对位置漂移
- **修复方案**：将地图图片、动画圆圈、标注点、文字全部移入同一个 `<svg>` 元素，共享统一 viewBox 坐标系
- **viewBox 精确校准**：viewBox 从 `600×600` → `600×524` → `600×547`，最终精确匹配地图原图真实宽高比（1380:1258），消除 X/Y 方向缩放比例不一致导致的偏移
- **坐标精确测量**：使用 Python + PIL 对地图原图进行像素级扫描，精确检测各水库发光点中心坐标（误差 < 1px），替代之前的目测估算
- **图片铺满方案**：`<image>` 标签改用 `preserveAspectRatio="none"` 铺满 viewBox，消除 `xMidYMid meet` 模式下图片两侧留白导致的坐标偏移

## [v1.4.0] — 2026-04-04

### 新增功能

**天气预警系统**

- 当 24h 降水量超过 20mm 或出现雷阵雨、暴雨、沙尘暴等灾害性天气时，整个标题区自动触发预警效果
- 支持四级预警等级：蓝色（一般）、黄色（注意）、橙色（警告）、红色（严重）
- 预警触发时标题区背景色、边框、文字颜色随等级变化，并显示预警标签和具体预警内容
- 标题区添加 CSS 呼吸灯动画，视觉上持续闪烁提示

## [v1.0.0] — 2026-03-24

### 初始版本发布

**监测信息驾驶舱**

- 三列布局：左侧信息面板（20%）+ 中部地图（60%）+ 右侧图表（20%）
- 左侧面板：水库选择下拉框、水库特征数据、实时监测指标、责任人信息（行政/技术/巡查）
- 中部地图：宜川县地图图片，叠加沿同心圆轨迹流动的亮线特效
- 右侧图表：近期降雨过程（柱状图）、库水情监测（折线图）、渗透压力监测（梯形剖面图）
- 顶部标题区：实时时钟（每秒更新）、天气信息展示

**运维信息驾驶舱**

- 三列布局与监测信息驾驶舱一致
- 左侧：区县运维情况（圆形进度条）、运维综合统计、问题处理统计、责任人信息
- 中部：顶部三个指标卡片 + 宜川县地图
- 右侧：区县水库运行信息、问题处理情况（饼状图）、库区水情监测、"查看工单"按钮

**工单管理弹窗**

- 点击"查看工单"按钮弹出工单列表
- 支持工单状态筛选（全部/待处理/处理中/已完成）
- 工单详情展示：编号、类型、水库、描述、负责人、创建时间、状态

**水库数据（本地模拟）**

- 四个水库：刘庄水库、钟楼寺水库、崖底水库、木头沟水库
- 每个水库包含：特征数据、实时监测指标、责任人信息（含真实姓名和联系电话）
- 切换水库时所有面板数据同步更新

**部署**

- 生产环境：`https://www.wisdomdance.cn/ycsl/`
